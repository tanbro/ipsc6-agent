# 收到振铃

座席被排队选中后，CTI服务器将呼叫座席，并向座席发送振铃通知，从而触发该事件。

- **Method**: `onRingReceived`

- **Params**:

    |  Argument  |       Type       | Default | Description |
    | ---------- | ---------------- | ------- | ----------- |
    | `ringInfo` | [振铃数据对象][] |         |             |

[振铃数据对象]: ../objects/ring_info.md
