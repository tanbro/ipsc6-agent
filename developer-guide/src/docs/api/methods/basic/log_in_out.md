# 登录与注销

## 登录

- **Method**: `logIn`

- **Params**:

    |  Argument   | Data Type | Default | Description |
    | ----------- | --------- | ------- | ----------- |
    | `workerNum` | `String`  | -       | 登录工号    |
    | `password`  | `String`  | -       | 登录密码    |

- **Result**: `null`

## 注销

- **Method**: `logOut`
- **Params**: 无
- **Result**: `null`
