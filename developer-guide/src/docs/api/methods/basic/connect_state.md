# 获取连接状态

- **Method**: `getConnectState`

- **Params**: 无

- **Result**: `null`

    | Argument |   Type    | Default |             Description             |
    | -------- | --------- | ------- | ----------------------------------- |
    | `index`  | `Integer` | -       | 连接所属的CTI服务器的索引值 |
    | `state`  | `Integer` | -       | [客户端-服务器连接状态][] 枚举值    |

[客户端-服务器连接状态]: ../../enums/client_connect_state.md
